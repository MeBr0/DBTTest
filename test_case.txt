Описание
Требуется разработать приложение TODO лист. TODO лист, это приложение в котором можно создавать, редактировать и удалять задачи. Задача представляет из себя простой многострочный текст. Пример приложения от сторонних разработчиков:

Google keep
Wunderlist

У пользователя должна быть возможность выполнить следующие действия

Создать задание с текстом;
Редактировать текст задания;
Выполнить задание;
Удалить задание;
Просмотреть список всех не удаленных задания;

Технические требования

UI должен быть разработан на Angular 7+;
Использовать Angular Material в качестве основной библиотеки стилей.
Серверную часть должен представлять Spring;
БД может быть любой на ваше усмотрение. (Вам нужно будет указать какую БД вы используете при отправке тестового задания)
